# The Hipcheck Developer's Guide to Hipcheck

This is a guide for _developers_ of Hipcheck, to document tools, processes,
and ideas that are relevant for someone contributing to Hipcheck. If you're
a user of Hipcheck and not contributing to its development, we recommend
you check out the [Quickstart Guide] or the [Complete Guide to Hipcheck].
